Installation

Essayer Clin — installateurs desktop (bêta)

Les installateurs Windows et macOS sont récents et peu testés. Je cherche des personnes curieuses et patientes pour essayer, rapporter les bugs et partager des retours — pas un produit fini prêt pour tout le monde.

  1. Sur GitHub Releases, télécharge l’app desktop pour ton OS et le zip de l’extension Chrome.
  2. Lance Clin (Windows : fichier portable — macOS : dézippe, puis clic droit → Ouvrir la première fois si macOS bloque l’app non signée).
  3. Chrome → Extensions → Mode développeur → Charger l’extension non empaquetée → dossier extrait du zip extension.
  4. Dans Clin, configure une clé API IA ou Ollama local si tu veux les brouillons assistés par IA.
  5. Teste sur LinkedIn, puis ouvre une issue ou écris-moi sur WhatsApp avec ce qui coince.

Fonctionnalités

Ce que Clin fait concrètement

Quatre piliers pour reprendre la main sur ton réseau LinkedIn, sans abonnement cloud ni copier-coller interminable.

📥

Capture

Sauvegarde automatiquement les profils et listes LinkedIn dans une base locale. Plus besoin de copier-coller manuellement.

🧹

Nettoyage

Score tes contacts, identifie les relations froides, les profils obsolètes. Reprends le contrôle de ta base de 500 à 16 000 relations.

✍️

Rédaction par lot

Prépare des séries de posts LinkedIn et des messages d'approche via un modèle d'IA de ton choix. Génère, révise, approuve — puis envoie toi-même.

⚙️

Automatisation optionnelle

Active l'autopilote uniquement si tu le choisis. Vitesse, caps, limites : tout est paramétrable. Tu restes aux commandes.

Public

Pour qui ?

✅ Clin est fait pour toi si :

  • Tu as une base LinkedIn de plusieurs centaines ou milliers de contacts
  • Tu veux reprendre la main sur ta gestion réseau sans payer un SaaS mensuel
  • Tu es prêt·e à tester une bêta desktop (bugs possibles) et à partager des retours
  • Tu as accès à une API d'IA (OpenAI, Mistral, Ollama local…) ou tu peux t'en passer au début

❌ Clin n'est pas encore pour toi si :

  • Tu attends un produit clé en main, stable et supporté comme un SaaS commercial
  • Tu ne veux rien installer (ni app desktop, ni extension Chrome)
  • Tu ne veux pas signaler de bugs ni participer à une bêta
Transparence

C'est open source

Clin est publié en open source sous licence MIT. Tu peux lire le code, le forker, l'adapter. C'est un terrain d'expérimentation partagé, pas un produit à vendre.
